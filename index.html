Объекты в JavaScript

Объект в JS, как структура хранения данных, может называться по разному в других языках программирования.

  • объект
  • словарь
  • карта, отображение (map)
  • хэш-таблица

Объект в JS — не соотвествует понятию объекта в ООП.
